# Kiss' ZSH Files

> My zsh configuration

## Installation

**1. Clone the repository**

```sh
git clone --recursive https://github.com/filipekiss/zsh ~/zshfiles
cd ~/zshfiles
```

**2. Link required folders**

```sh
make link
```

**3. Install Plugins**

```sh
make install-plugins
```

**3b. Update Plugins**

```sh
make update-plugins
```
